# Insurance Cost Data Analytics

Exploratory data analysis and feature engineering on insurance cost data.

[![CI](https://github.com/Tabasiarash/insurance-cost-analytics/actions/workflows/ci.yml/badge.svg)](https://github.com/Tabasiarash/insurance-cost-analytics/actions)

## Overview

Analytics project performing statistical analysis and visualization on an insurance dataset. Explores relationships between patient demographics, lifestyle factors, and insurance costs.

## Requirements

- Pandas, NumPy
- Matplotlib, Seaborn
- scikit-learn
- Jupyter Notebook

## Usage

Open `Data Analytics for Insurance Cost Data Set.ipynb` in Jupyter and run all cells.
